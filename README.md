## 核心设计

所有规则都实现 `Rule` trait：

```rust
pub trait Rule: Send + Sync {
    fn metadata(&self) -> &RuleMetadata;  // 规则元数据
    fn category(&self) -> RuleCategory;   // 规则分类
    fn validate(&self, context: &str) -> RuleResult<bool>; // 验证
    fn explain(&self) -> String;          // 规则说明
}
```
